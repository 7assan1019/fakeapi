‚ö° Client-Side Features

This is a static API that returns complete datasets. Use JavaScript to filter, sort, and paginate the data on the client-side.

Client-Side Filtering

Filter data using JavaScript after fetching the complete dataset

// Get all hotels, then filter
fetch('/api/hotels')
  .then(response => response.json())
  .then(hotels => {
    const fiveStarHotels = hotels.filter(hotel => hotel.stars === 5);
    const affordableHotels = hotels.filter(hotel => hotel.price <= 200);
  })

Client-Side Sorting

Sort results using JavaScript's sort() method

// Get all products, then sort
fetch('/api/products')
  .then(response => response.json())
  .then(products => {
    const sortedByPrice = products.sort((a, b) => a.price - b.price);
    const sortedByRating = products.sort((a, b) => b.rating - a.rating);
  })

Client-Side Search

Search through data using JavaScript after fetching

// Get all products, then search
fetch('/api/products')
  .then(response => response.json())
  .then(products => {
    const searchTerm = 'laptop';
    const results = products.filter(product => 
      product.name.toLowerCase().includes(searchTerm)
    );
  })

Client-Side Pagination

Implement pagination using JavaScript array methods

// Get all users, then paginate
fetch('/api/users')
  .then(response => response.json())
  .then(users => {
    const page = 1;
    const limit = 10;
    const start = (page - 1) * limit;
    const paginatedUsers = users.slice(start, start + limit);
  })

Data Relationships

Combine data from multiple endpoints to create relationships

// Get users and their posts
Promise.all([
  fetch('/api/users'),
  fetch('/api/posts')
]).then(responses => Promise.all(responses.map(r => r.json())))
.then(([users, posts]) => {
  const usersWithPosts = users.map(user => ({
    ...user,
    posts: posts.filter(post => post.userId === user.id)
  }));
})
