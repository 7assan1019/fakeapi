⚡ Advanced Features

Discover the power of our API with these advanced capabilities

Filtering

Filter data by any field with various operators

GET http://localhost:3000/api/hotels?stars=5&price_lte=200

GET http://localhost:3000/api/products?rating_gte=4.5

GET http://localhost:3000/api/users?address.city=Cairo

Sorting

Sort results by any field in ascending or descending order

GET http://localhost:3000/api/products?_sort=price&_order=asc

GET http://localhost:3000/api/jobs?_sort=salary&_order=desc

GET http://localhost:3000/api/hotels?_sort=rating&_order=desc

Search

Full-text search across all text fields

GET http://localhost:3000/api/products?q=laptop

GET http://localhost:3000/api/hotels?q=resort

GET http://localhost:3000/api/jobs?q=developer

Pagination

Control the number of results and navigate pages

GET http://localhost:3000/api/users?_page=1&_limit=10

GET http://localhost:3000/api/products?_start=20&_limit=5

GET http://localhost:3000/api/hotels?_page=2&_limit=15

Nested Routes

Access related data through nested endpoints

GET http://localhost:3000/api/posts/1/comments

GET http://localhost:3000/api/users/1/posts

GET http://localhost:3000/api/products/1/reviews

Complex Queries

Combine multiple features for powerful queries

GET http://localhost:3000/api/apartments?bedrooms_gte=2&price_lte=2000&_sort=price&_order=asc&_page=1&_limit=10

GET http://localhost:3000/api/hotels?stars=5&amenities_like=Pool&_sort=rating&_order=desc
