Getting Started

Welcome to Fake API! This is a free, public REST API service designed to help developers build and test their applications with realistic data.

Base URL: http://localhost:3000/api

What is Fake API?

Fake API provides 21 different resources with over 500 realistic data items. It's perfect for:

  • Frontend development and testing
  • Learning REST APIs
  • Prototyping applications
  • API integration testing

Quick Example

Request

fetch('http://localhost:3000/api/users/1')
  .then(response => response.json())
  .then(json => console.log(json))

API Endpoints

All endpoints support standard HTTP methods: GET, POST, PUT, DELETE, and PATCH.

Core Endpoints

GEThttp://localhost:3000/api
API information and available endpoints

GEThttp://localhost:3000/health
Server health check

GEThttp://localhost:3000/api/users
Get all users

GEThttp://localhost:3000/api/users/1
Get user by ID

POSThttp://localhost:3000/api/users
Create a new user

PUThttp://localhost:3000/api/users/1
Update user by ID

DELETEhttp://localhost:3000/api/users/1
Delete user by ID

Authentication

No authentication required! All endpoints are publicly accessible and free to use.

Querying Data

You can query data using various parameters to filter, sort, and paginate results.

Basic Query Structure

http://localhost:3000/api/resource?parameter=value¶meter2=value2

Filtering

Filter data by specific field values using various operators.

Exact Match

GET http://localhost:3000/api/hotels?stars=5

Get all 5-star hotels

Greater Than

GET http://localhost:3000/api/products?price_gte=100

Products with price ≥ $100

Less Than

GET http://localhost:3000/api/products?price_lte=500

Products with price ≤ $500

Array Contains

GET http://localhost:3000/api/hotels?amenities_like=Pool

Hotels with pool amenity

Nested Object

GET http://localhost:3000/api/users?address.city=Cairo

Users from Cairo

Multiple Filters

GET http://localhost:3000/api/apartments?bedrooms_gte=2&price_lte=2000

2+ bedrooms under $2000

Pagination

Control the number of results returned and navigate through pages.

Limit Results

GET http://localhost:3000/api/users?_limit=10

Get only 10 users

Page Navigation

GET http://localhost:3000/api/users?_page=2&_limit=10

Get users 11-20

Skip Results

GET http://localhost:3000/api/users?_start=20&_limit=10

Skip first 20, get next 10

Sorting

Sort results by any field in ascending or descending order.

Ascending Order

GET http://localhost:3000/api/products?_sort=price&_order=asc

Sort products by price (low to high)

Descending Order

GET http://localhost:3000/api/hotels?_sort=rating&_order=desc

Sort hotels by rating (high to low)

Multiple Fields

GET http://localhost:3000/api/jobs?_sort=salary,title&_order=desc,asc

Sort by salary desc, then title asc

Search

Search across all text fields in a resource.

Text Search

GET http://localhost:3000/api/products?q=laptop

Search for products containing "laptop"

Hotel Search

GET http://localhost:3000/api/hotels?q=resort

Search for hotels containing "resort"

Job Search

GET http://localhost:3000/api/jobs?q=developer

Search for jobs containing "developer"

Error Handling

All errors return appropriate HTTP status codes and helpful error messages.

Common Error Responses

404 Not Found

GET http://localhost:3000/api/nonexistent

Resource doesn't exist

400 Bad Request

POST http://localhost:3000/api/users

Invalid request data

500 Server Error

Any endpoint

Internal server error

Error Response Format

{
  "error": "Endpoint not found",
  "message": "The endpoint /api/nonexistent does not exist",
  "availableEndpoints": "/api"
}
